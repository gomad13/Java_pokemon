# Projet Pokemon - Combat

Mini-jeu de combat Pokemon en console, developpe en Java dans le cadre du projet de premiere annee.

## Description

Le joueur compose une equipe de 3 Pokemon et affronte une IA qui a aussi sa propre equipe.
Le combat se fait au tour par tour, avec un systeme de types comme dans le vrai jeu.

## Technologies utilisees

- Java
- SQLite (base de donnees)
- JDBC (connexion a la base)

## Comment lancer le jeu

A completer une fois le projet termine.
